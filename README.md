## Overview

AlchemyX is an autonomous AI agent built on IBM watsonx Orchestrate (ABM Watson X-Architecture) that eliminates information fragmentation across enterprise systems:

- **Data Synthesizer** - retrieves and reconciles data from CRM, ERP, Messaging, Databases, and Email into one coherent narrative
- **Trust Engine** - provides 100% source attribution (e.g., [Source: CRM]) to ensure transparency and prevent hallucinations

Built with Google AI Studio frontend, powered by IBM watsonx, and designed for enterprise-scale data intelligence.

## Features

- Synthesizes fragmented data into one conclusive, actionable report
- Provides source-attributed answers to eliminate misinformation
- Detects data conflicts and quality risks during synthesis
  - Automatically flags high-priority tasks for human review
  - Generates tasks in Central Task Manager
- Career constellation visualization showing primary and alternative paths
- Converts wasted search time into high-confidence business insights
- Enterprise-scale design bridging data silos across multiple systems

## Tech Stack

- **Backend / Agent:** IBM watsonx Orchestrate (ABM Watson X-Architecture)
- **Frontend:** Google AI Studio
- **Languages:** TypeScript, HTML, Tailwind CSS
